Hallo sedang belajar git neh..

setelah sempet error berhasil push dengan perintah berikut:
git remote add origin https://github.com/indra2878/Belajar_Git.git
git push -u origin master

lalu sinkronisasi dengan :
git pull origin master

bila  melakukan perubahan isi file teks (di lokal):
save dulu perubahan tersebut pada file teks.
git status (terlihat file_teks modified dgn warna merah belum masuk stage (antrian commit)
git add file_teks.txt (memindah file revisi ke stage untuk dicommit)
git commit -m "menambah tulisan pada file_teks.txt"
git push -u origin master

lalu sinkronisasi:
git pull origin master
